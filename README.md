# Azeez Fit AI - Halal Nutrition & Weight-Loss Coach

A lightweight, installable web app for tracking weight, calories, and protein intake using common halal hawker centre foods. Built for iPhone as a Progressive Web App (PWA).

## Features

✅ **Weight & Progress Tracking**
- BMI calculation
- Progress visualization
- Weight, waist, and steps logging

✅ **Nutrition Logging**
- Pre-loaded halal hawker food database with calorie & protein estimates
- Photo-based food logging with manual calorie entry
- Barcode scanning using Open Food Facts API
- Daily calorie and protein summaries

✅ **Offline-First**
- Service Worker for offline functionality
- All data stored locally in browser
- Optional CSV export

✅ **iOS Installation**
- One-tap home screen installation
- Standalone app experience
- No app store required

## Quick Start

### Install on iPhone

1. **Open in Safari**
   ```
   https://azeez0071-ctrl.github.io/Health-fit-halal
   ```

2. **Share → Add to Home Screen**
   - Tap the Share button
   - Select "Add to Home Screen"
   - Tap Add

3. **Launch from home screen** – The app runs offline after first load

## Daily Targets

- **Calories:** 1,600–1,900 kcal/day
- **Protein:** 90–110 g/day
- **Water:** 2–2.5 L/day
- **Activity:** 8,000+ steps

## Hawker Food Database

Pre-loaded foods with calorie & protein estimates

## Technology

- **Frontend:** Vanilla JavaScript, HTML5, CSS3
- **Storage:** Browser localStorage
- **PWA:** Manifest + Service Worker
- **Barcode API:** Open Food Facts
- **Offline:** Works without internet after first load

## Privacy & Data

✅ **All data stays in your browser**
- No server uploads
- No tracking
- Optional CSV export for backup
